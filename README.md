# Kalshi Market Graph
To compile and run with Maven (pom.xml supports this)!

mvn compile
mvn exec:java

To compile and run without maven:

javac $(find src/main/java -name "*.java")
java -cp src/main/java edu.upenn.nets1500.kalshi.App

Refer to the readme.txt file for detailed project overview and the user manual pdf for how to use program.
